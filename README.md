# PriceRadar - India's Best Price Comparison Platform

Live Website: https://priceradar-one.vercel.app/

## What is PriceRadar?
PriceRadar is a full-stack price comparison website that scans 34+ Indian shopping sites in real time and shows you the best price for any product. Never overpay again!

## Features
- Real-time Price Comparison across Amazon, Flipkart, Myntra, Meesho, Croma and 30+ more
- Product Images from all shopping sites
- Price Filters by price range and rating
- Grid and List View for search results
- Product Detail Page with direct buy links
- Lowest Price highlighted in green
- You Save Calculator
- User Authentication with email and Google
- Price Alerts via email
- Wishlist to save products
- Shop by Category

## Tech Stack
- Frontend: Next.js 15, React 19, TypeScript
- Backend: Next.js API Routes
- Database: PostgreSQL on Supabase
- Cache: Redis on Upstash
- Authentication: NextAuth.js
- Price Data: SerpAPI
- Deployment: Vercel

## Getting Started
1. Clone the repo
2. Run npm install
3. Create .env file with your keys
4. Run npx prisma db push
5. Run npm run dev
6. Open http://localhost:3000
